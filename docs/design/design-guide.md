# 设计风格指南 — Stripe 风格

## 风格来源
Stripe 金融级设计语言，适合企业后台管理系统。

## 设计 Token

### 色彩系统

| Token | 值 | 用途 |
|:------|:---|:-----|
| `--color-primary` | `#533afd` | 主色：按钮、链接 |
| `--color-primary-deep` | `#4434d4` | 主色悬停 |
| `--color-primary-press` | `#2e2b8c` | 主色按下 |
| `--color-ink` | `#0d253d` | 正文文字（深海军蓝） |
| `--color-ink-mute` | `#64748d` | 辅助文字 |
| `--color-canvas` | `#ffffff` | 页面底色 |
| `--color-canvas-soft` | `#f6f9fc` | 浅色面板底色 |
| `--color-hairline` | `#e3e8ee` | 分割线、边框 |
| `--color-ruby` | `#ea2261` | 删除/危险操作 |

### 字体
- 主字体：Inter（Sohne 商业字体替代）
- 字号：正文 14px，标题 18-22px，大标题 26-32px

### 圆角
- 按钮：圆角 pill（9999px）
- 卡片：12px
- 表单输入框：6px

### 间距基准
- 基础间距：8px
- 卡片内边距：24px
- 大区块间距：32-64px

### 阴影
- 卡片阴影：`0 1px 3px rgba(0,55,112,0.08)`
- 浮动阴影：`0 8px 24px rgba(0,55,112,0.08)`

## 组件样式

### 按钮
- 主按钮：靛蓝填充 + 白色文字 + pill 圆角
- 次按钮：白色底 + 靛蓝边框
- 危险按钮：红色填充

### 表格
- 白色卡片容器 + 圆角 12px
- 表头浅底 + 大写标题
- 行悬停高亮
- 无边框网格，分割线细而淡

### 卡片
- 白底 + 12px 圆角 + 细边框 + 微阴影
- 统计卡片：大数字（32px） + 小标签

### 表单
- 白底 + 6px 圆角 + 淡蓝边框
- 聚焦时边框变靛蓝 + 光晕

### 状态标签
- 圆角 pill
- 颜色根据状态：黄色=待审批、绿色=已通过、红色=已驳回

## 设计原则
1. 留白充足，信息密度适中
2. 主色（靛蓝）只用于关键操作，不滥用
3. 表格和卡片是数据展示的核心载体
4. 所有页面样式从 design.css 中引用，不自创
