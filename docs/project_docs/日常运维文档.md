# 项目工时管理系统 - 日常运维文档

版本：V1.0　　日期：2026-06-17

---

## 1. 服务管理

### 1.1 启动服务

```bash
cd /home/sukai/workspace/project_time_manager

# 开发环境
mvn spring-boot:run

# 生产环境
java -jar target/time-manager-1.0.0.jar --spring.profiles.active=prod

# 后台运行
nohup java -jar target/time-manager-1.0.0.jar > app.log 2>&1 &
```

### 1.2 停止服务

```bash
ps aux | grep time-manager
kill {PID}
```

### 1.3 查看状态

```bash
lsof -i :8080
curl -s -o /dev/null -w "%{http_code}" http://localhost:8080/api/auth/captcha
tail -f app.log
```

## 2. 日志管理

```bash
tail -f app.log
grep -i "error|exception" app.log
grep -i "started|listening" app.log
```

## 3. 备份恢复

```bash
# 备份
mysqldump -u time_mgr -p project_time_manager > backup/ptm_$(date +%Y%m%d).sql

# 恢复
mysql -u time_mgr -p project_time_manager < backup/ptm_20260617.sql
```

## 4. 常见问题

| 问题 | 原因 | 处理方法 |
|:-----|:-----|:---------|
| 端口被占用 | 其他进程占用8080 | lsof -i :8080 -> kill {pid} |
| 数据库连接失败 | MySQL未启动或密码错误 | systemctl status mysql |
| Token验证失败 | JWT密钥配置不一致 | 检查application.yml中jwt.secret |
