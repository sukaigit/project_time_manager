# 项目工时管理系统 - 用户操作手册

版本：V1.0　　日期：2026-06-17

---

## 1. 系统概述

项目工时管理系统是公司研发与交付中心的工时数字化管理平台，支持工时填报、审批、统计报表等功能，为月度/年度绩效考核提供数据支撑。

系统地址：http://localhost:8080

## 2. 登录与退出

### 2.1 登录

1. 打开浏览器访问系统地址
2. 输入用户名和密码
3. 输入图形验证码
4. 点击「登录」按钮

首次登录会弹出修改密码对话框，需修改密码后才能继续使用。

### 2.2 退出

点击导航栏右上角的退出按钮，清除登录信息并跳转到登录页。

### 2.3 修改密码

1. 点击导航栏右上角用户名
2. 选择「修改密码」
3. 输入旧密码和新密码（>=6位）
4. 点击「确认修改」

## 3. 首页大屏

登录后进入首页大屏，展示内容包括：

- 今日概况：今日提交人数、总工时
- 我的待办：待审批条数（仅PM/部门经理）
- 本月完成率：按月度和部门/项目维度展示
- 项目工时分布：饼图展示各项目工时占比
- 整体概览：部门总人数、总工时、人均工时

## 4. 工时填报

### 4.1 填报工时

1. 进入「我的工时」页面
2. 点击「填报工时」按钮
3. 选择日期、项目、任务模块
4. 填写工时（小时，支持小数点）和工作内容
5. 点击「保存」

注：普通用户填报时，模块累计工时达到预算上限后不可再报。

### 4.2 修改/删除工时

- 未审批的工时可直接修改或删除
- 已审批的工时不可修改或删除

### 4.3 查看工时列表

筛选条件：项目、日期范围
查看维度：个人/项目

## 5. 工时审批

### 5.1 待审批列表

1. 进入「工时审批」页面
2. 查看待审批的工时记录
3. 点击「通过」或「驳回」

### 5.2 批量审批

1. 勾选多条记录
2. 点击「批量通过」或「批量驳回」
3. 填写审批意见（可选）

注：普通用户的工时由所属项目的项目经理审批，项目经理的工时由部门经理审批。

## 6. 项目管理

### 6.1 项目管理

1. 进入「项目管理」页面
2. 查看项目列表
3. 新增/编辑/删除项目

项目经理只能从PM角色用户中选择。

### 6.2 任务模块管理

在项目详情页中管理任务模块，新增模块时需填写预估工时预算。

## 7. 统计报表

### 7.1 个人统计

按年月筛选，查看个人总工时、各项目工时明细。

### 7.2 项目统计

按年月筛选，查看项目总工时、各模块实报vs预算对比。

### 7.3 部门统计

按年月筛选，查看部门总工时、人均工时、各项目占比。

### 7.4 导出Excel

点击「导出」按钮，下载当前筛选条件下的数据。

## 8. 系统管理（仅管理员）

### 8.1 用户管理

- 新增用户：填写信息，密码自动设为123456
- 编辑/删除用户
- 重置用户密码
- 启用/禁用用户账号

### 8.2 操作日志

查看所有用户的操作日志，支持按操作人、操作类型、日期范围筛选。

## 9. 常见问题

| 问题 | 解决方法 |
|:-----|:---------|
| 忘记密码 | 联系系统管理员重置密码（重置为123456） |
| 无法登录 | 检查是否输入了正确的验证码 |
| 工时无法提交 | 检查是否已达预算上限或账号被禁用 |
| 找不到审批记录 | 确认是否有对应角色的审批权限 |
