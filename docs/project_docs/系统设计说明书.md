版本：V1.0　　日期：2026-06-17

## 1. 系统架构设计

### 1.1 整体架构

系统采用前后端分离的三层架构：前端为原生 HTML/JS 单页应用，后端为 Spring Boot REST API，数据层使用 MySQL 8.0。

| 层级 | 职责 | 技术选型 |
|:-----|:-----|:---------|
| 展示层 | 用户交互、页面路由 | 原生 HTML + CSS（Bootstrap5）+ JS |
| 应用层 | 业务逻辑、API | Spring Boot 3.2.5 + MyBatis |
| 数据层 | 数据持久化 | MySQL 8.0 |

### 1.2 架构层次

```
+--------------------------------------------------------------+
|                    展示层（原生 HTML/JS）                       |
|  登录页    管理后台    工时模块    数据看板                      |
+--------------------------------------------------------------+
|                  API 层（Spring Boot REST）                    |
|  auth/*  users/*  projects/*  modules/*  work-hours/*          |
|  approvals/*  dashboard/*  reports/*  logs                     |
+--------------------------------------------------------------+
|                Service 层（业务逻辑）                          |
|  AuthService  UserService  ProjectService  ModuleService       |
|  WorkHourService  ApprovalService  DashboardService            |
|  ReportService  LogService                                     |
+--------------------------------------------------------------+
|                Mapper 层（MyBatis XML）                        |
|  UserMapper  ProjectMapper  TaskModuleMapper  WorkHourMapper   |
|  ApprovalMapper  OperationLogMapper                            |
+--------------------------------------------------------------+
|                       MySQL 数据库                              |
+--------------------------------------------------------------+
```

### 1.3 技术栈

| 层级 | 技术 | 版本 | 用途 |
|:-----|:-----|:-----|:-----|
| 前端 | 原生 HTML + CSS + JS | - | 页面渲染、交互 |
| CSS框架 | Bootstrap 5 | - | UI 组件库 |
| 图表 | ECharts | - | 大屏和报表可视化 |
| 后端 | Spring Boot | 3.2.5 | REST API |
| ORM | MyBatis | 3.0.3 | 数据访问 |
| 认证 | JWT + Spring Security | - | 无状态鉴权 |
| 验证码 | easy-captcha | 1.6.2 | 图形验证码 |
| 数据库 | MySQL | 8.0 | 数据存储 |
| Excel | Apache POI | 5.2.5 | 报表导出 |

## 2. 模块设计

### 2.1 Auth 认证模块

| 属性 | 内容 |
|:-----|:-----|
| 模块名 | Auth |
| 职责 | 用户认证鉴权、验证码生成、密码管理 |
| 涉及文件 | AuthController, AuthService, JwtUtil, JwtAuthFilter, SecurityConfig |

| 模块名 | User |

| 技术点 | 逻辑删除、角色枚举转换、firstLogin状态管理 |

| 模块名 | Project |
| 职责 | 项目CRUD、项目经理分配、状态管理 |

| 模块名 | Approval |

| 模块名 | Dashboard |
| 职责 | 数据聚合展示、ECharts图表渲染 |

### 2.8 Report 统计报表

## 4. 安全方案

### 4.1 认证与授权

| 机制 | 实现方式 | 说明 |
|:-----|:---------|:-----|
| 身份认证 | JWT | Bearer Token，24h过期 |
| 授权模型 | RBAC | USER/PM/DEPT_MANAGER/ADMIN 四种角色 |
| 权限控制 | SecurityConfig | URL按角色配置访问规则 |

### 4.2 安全防护

| 风险 | 防护措施 |
|:-----|:---------|
| SQL注入 | MyBatis参数化查询 #{}，无${}拼接 |
| XSS | API返回JSON数据，innerHTML仅数据显示（非用户输入） |
| 密码泄露 | BCrypt加密存储，salt rounds >= 10 |
| 暴力登录 | 图形验证码防暴力破解 |
