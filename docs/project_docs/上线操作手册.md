# 项目工时管理系统 - 上线操作手册

版本：V1.0　　日期：2026-06-17

---

## 1. 环境要求

### 1.1 软件环境

| 软件 | 版本 | 用途 |
|:-----|:-----|:-----|
| JDK | 17+ | Spring Boot 运行时 |
| MySQL | 8.0 | 数据库 |
| Maven | 3.8+ | 项目构建 |

### 1.2 硬件环境

| 环境 | CPU | 内存 | 磁盘 |
|:-----|:-----|:-----|:-----|
| 最低 | 2核 | 4GB | 20GB |
| 推荐 | 4核 | 8GB | 50GB |

## 2. 部署步骤

```bash
# 1. 克隆代码
git clone https://github.com/sukaigit/project_time_manager.git
cd project_time_manager

# 2. 初始化数据库
mysql -u root -p < sql/init.sql

# 3. 创建应用用户
mysql -u root -p -e "
  CREATE USER IF NOT EXISTS 'time_mgr'@'localhost' IDENTIFIED BY 'TimeMgr@2026';
  GRANT SELECT, INSERT, UPDATE, DELETE ON project_time_manager.* TO 'time_mgr'@'localhost';
  FLUSH PRIVILEGES;
"

# 4. 编译打包
mvn clean package -DskipTests

# 5. 启动
java -jar target/time-manager-1.0.0.jar

# 6. 验证
curl http://localhost:8080/api/auth/captcha
```

## 3. 配置说明

| 配置项 | 默认值 | 说明 |
|:-------|:-------|:-----|
| server.port | 8080 | 服务端口 |
| spring.datasource.url | jdbc:mysql://localhost:3306/project_time_manager | 数据库连接 |
| spring.datasource.username | time_mgr | 数据库用户 |
| jwt.secret | YourSuperSecretKeyForJWT... | JWT密钥（生产需更换） |
| jwt.expiration | 86400000 | Token过期时间(ms) 24h |

## 4. 首次使用

- 预置管理员：admin / 123456
- 首次登录后系统强制修改密码
- 访问地址：http://{服务器IP}:8080
